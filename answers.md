Question: What is an Agent?
Answer: An Agent is an application that is installed on a server/machine. It’s purpose is to collect data from the machine it is installed on and feed this back to DataDog.

Question: What is the difference between a timeboard and a screenboard?
Answer: Timboards and Screenboards are type of Dashboards on the Datadog interface. A timeboard contains a series for graphs that are based on a certain time and each graph is displayed as a matrix. A screenboard allows more customization and can be shared in read only format.
